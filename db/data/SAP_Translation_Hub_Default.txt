SAP Translation Hub
SAP Translation Hub lets you translate your UI texts or documents by using a repository of SAP-approved translations and terminology as well as machine translation. You can access the translation resources by consuming a range of API methods or by using integrated workflow scenarios on a UI. Furthermore, developers can get suggestions for their source texts during the development phase, which improves accuracy and saves time.Translate UI texts and documents and get UI text suggestions during development.
Service Plan: Default (SAP Translation Hub enables customers and partners to satisfy the demands of a global market by translating the short texts of products into additional languages. Translate short texts of existing products into additional languages by using multilingual texts stored in an SAP HANA database. The translation workflow capabilities of SAP Translation Hub enable you to translate the properties files in HTML5 applications that are built on SAP Business Technology Platform at the push of a button.)

SAP Task Center
The SAP Task Center service enables integration with SAP and non-SAP applications to provide a single entry point for end users to access all their assigned tasks. The tasks can be accessed by end users through the SAP Task Center Web app, SAP Mobile Start app, and SAP Start.SAP Task Center helps you integrate tasks into a central solution.
